# 🛒 QazynaDelivery — Telegram-бот для доставки продуктов

## Описание

Telegram-бот для автоматизации приёма заказов продуктов от жителей жилого комплекса. Разработан по техническому заданию для магазина "QazynaDelivery".

## Возможности

### 👤 Для клиентов:
- 🛒 **Каталог товаров** с категориями (Молочное, Хлеб, Овощи/Фрукты, Мясо, Бытовая химия, Напитки)
- 🛍 **Корзина** с управлением количеством товаров
- 📦 **Оформление заказа** с выбором адреса, времени доставки и способа оплаты
- 👤 **Личный кабинет** с историей заказов и сохранёнными данными
- 📍 **Сохранение адреса** для повторных заказов

### 🔧 Для администратора:
- 🆕 **Мгновенные уведомления** о новых заказах
- 📦 **Управление статусами** заказов (Новый → В работе → Отправлен → Доставлен)
- ❌ **Возможность отмены** заказов
- 📋 **Просмотр всех заказов** по статусам

## Технологии

- **Python 3.9+**
- **aiogram 3.x** — асинхронный фреймворк для Telegram Bot API
- **SQLite** — локальная база данных
- **aiosqlite** — асинхронная работа с SQLite

## Установка

### 1. Клонирование и установка зависимостей

```bash
cd qazyna_bot
pip install -r requirements.txt
```

### 2. Настройка окружения

```bash
cp .env.example .env
```

Отредактируйте файл `.env`:
- `BOT_TOKEN` — токен от @BotFather
- `ADMIN_ID` — ваш Telegram ID (узнать через @userinfobot)

### 3. Запуск

```bash
python bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота, главное меню |
| `/history` | История заказов |
| `/profile` | Личный кабинет |
| `/admin` | Панель администратора (только для ADMIN_ID) |

## Демо-данные

При первом запуске бот автоматически создаёт демо-каталог с 25 товарами в 6 категориях. Вы можете заменить их на реальные данные, отредактировав функцию `_seed_demo_data()` в `database.py`.

## База данных (SQLite)

### Таблицы:
- **users** — пользователи
- **categories** — категории товаров
- **products** — товары
- **cart_items** — корзины (временное хранение)
- **orders** — заказы
- **order_items** — позиции заказов
